不。项目 .env 仅原生支持 两个 提供商：

Provider Env 变量
OpenAI (及兼容API) OPENAI_API_KEY, OPENAI_BASE_URL
Anthropic ANTHROPIC_API_KEY

所有其他厂商（DeepSeek、阿里Qwen、Google Gemini、Meta Llama、Mistral、xAI、Cohere、Databricks、上海AI Lab、澜舟科技等）没有专用的 env 变量。

变通方法：如果厂商提供 OpenAI 兼容的 API 端点（如 DeepSeek、Groq、OpenRouter），可将 OPENAI_BASE_URL 指向其地址来使用。否则需修改 src/agent/models.py:resolve_model() 添加新 provider 分支。
