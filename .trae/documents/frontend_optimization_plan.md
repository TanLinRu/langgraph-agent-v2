# 前端页面展示效果和动画优化计划

## 一、项目现状分析

### 1.1 当前已实现的动画效果

| 组件 | 现有动画 | 文件位置 |
|------|----------|----------|
| App.vue | 背景 orbs 浮动动画 | ui/src/App.vue |
| AgentMessage.vue | 消息进入、头像脉冲、徽章脉冲、光标闪烁 | ui/src/components/message/AgentMessage.vue |
| ThinkingPanel.vue | 思考点动画、chunk 进入动画 | ui/src/components/ThinkingPanel.vue |
| Sidebar.vue | 侧边栏滑入、会话悬停、删除按钮滑入 | ui/src/components/Sidebar.vue |

### 1.2 现有主题系统

- 深色/浅色主题切换
- CSS 变量定义约 60+ 个
- 玻璃拟态设计风格

## 二、优化目标

提升用户体验，使界面更加流畅和现代化，包括：

1. **微交互动画**：按钮悬停、卡片悬停、过渡效果
2. **消息动画**：更流畅的打字机效果、消息气泡动画
3. **视觉反馈**：状态变化的动画提示
4. **布局过渡**：平滑的面板展开/收起动画

## 三、优化方案

### 3.1 消息气泡增强

**目标文件**: `ui/src/components/message/AgentMessage.vue`

**优化内容**:
- 添加消息气泡的悬停效果（阴影、缩放）
- 增强打字机光标动画（更自然的闪烁）
- 添加文件引用的悬停动画
- 工具调用卡片的进入动画

### 3.2 思考面板增强

**目标文件**: `ui/src/components/ThinkingPanel.vue`

**优化内容**:
- 添加面板展开/收起的平滑过渡
- 增强思考中状态的视觉反馈
- 添加滚动条样式优化

### 3.3 侧边栏增强

**目标文件**: `ui/src/components/Sidebar.vue`

**优化内容**:
- 会话列表项的更丰富悬停效果
- 添加会话切换的过渡动画
- 优化搜索框的聚焦状态

### 3.4 输入框增强

**目标文件**: `ui/src/components/InputBar.vue`

**优化内容**:
- 添加输入框聚焦/失焦动画
- 增强发送按钮的交互动画
- 添加命令建议的下拉动画

### 3.5 背景动画增强

**目标文件**: `ui/src/App.vue`

**优化内容**:
- 增强 orbs 的层次和深度感
- 添加动态背景渐变效果
- 优化主题切换动画

## 四、实施步骤

| 步骤 | 任务 | 预计改动 |
|------|------|----------|
| 1 | 优化 AgentMessage 组件动画 | 消息气泡悬停效果、文件引用动画 |
| 2 | 优化 ThinkingPanel 组件 | 展开/收起动画、状态反馈 |
| 3 | 优化 Sidebar 组件 | 会话项动画、搜索框增强 |
| 4 | 优化 InputBar 组件 | 输入框动画、按钮效果 |

## 五、风险与注意事项

1. **性能考虑**：过多动画可能影响性能，需合理使用 `will-change` 和 GPU 加速
2. **兼容性**：确保动画在不同浏览器中表现一致
3. **降级方案**：支持 `prefers-reduced-motion` 用户偏好

## 六、验收标准

1. 所有新增动画流畅无卡顿
2. 悬停状态有明显视觉反馈
3. 过渡效果平滑自然
4. 主题切换保持一致性
5. 支持减少动画偏好设置
